Project on console

How to set up VS Code: Download and install VS Code. Create a new file. See an overview of the user interface. Install support for your favorite programming language. Change your keyboard shortcuts and easily migrate from other editors using keymap extensions. Customize your editor with themes.

How to open console screen in VS Code: Use the Ctrl+` keyboard shortcut with the backtick character. Use the View > Terminal menu command. From the Command Palette (Ctrl+Shift+P), use the View: Toggle Integrated Terminal command.
